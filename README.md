# Math Toolkit

Math Toolkit is a comprehensive mathematics library written in Python. It offers a wide range of functions from basic arithmetic and combinatorial calculations to number analysis and geometric operations.

## 🚀 Features

- ✅ Arithmetic operations (addition, subtraction, multiplication, division, modulus, exponentiation, square root)
- ✅ Combinatorial calculations (permutation, combination)
- ✅ Number analyses (prime number, perfect number, Armstrong number, even/odd check, digit sum)
- ✅ Geometric calculations (hypotenuse, Pythagorean triple check)
- ✅ Number system conversions (binary, octal, hexadecimal)
- ✅ Special operations (logarithm, absolute value, GCD-LCM, number reversal)

## 📦 Installation

```bash
git clone https://github.com/andacberkaye/math-toolkit.git
cd math-toolkit
```

## 🧪 Usage

```python
from mathOperations import Addition, is_prime, Hypotenuse, Binary

print(Addition(5, 10, 15))        # 30
print(is_prime(7))                # 7 is prime
print(Hypotenuse(3, 4))           # Hypotenuse: 5.0
print(Binary(10))                 # 1010
```

## 📘 Functions

| Function                  | Description                                          |
| ------------------------- | ---------------------------------------------------- |
| `Addition(*args)`         | Adds numbers                                         |
| `Subtraction(*args)`      | Subtracts numbers in order                           |
| `Multiplication(*args)`   | Multiplies numbers                                   |
| `Division(a, b)`          | Performs division a / b                              |
| `Exponents(a, b)`         | Calculates a to the power of b                       |
| `Roots(a)`                | Calculates the square root of a                      |
| `Factorial(a)`            | Calculates factorial of a (a!)                       |
| `Mean(*args)`             | Calculates average                                   |
| `Mod(a, b)`               | Calculates modulus                                   |
| `absolute_value(a)`       | Returns absolute value                               |
| `Permutation(a, b)`       | Calculates permutation                               |
| `Combination(a, b)`       | Calculates combination                               |
| `even_odd(a)`             | Checks if number is even or odd                      |
| `is_prime(a)`             | Checks if number is prime                            |
| `perfect_number(a)`       | Checks if number is perfect                          |
| `armstrong_number(a)`     | Checks if number is Armstrong number                 |
| `digits_sum(a)`           | Returns sum of digits                                |
| `pythagoras_three(a,b,c)` | Checks Pythagorean triple                            |
| `Hypotenuse(a, b)`        | Calculates hypotenuse                                |
| `log_base(x, base)`       | Calculates logarithm with given base                 |
| `Hexadecimal(a)`          | Converts number to hexadecimal                       |
| `Octal(a)`                | Converts number to octal                             |
| `Binary(a)`               | Converts number to binary                            |
| `reverse_number(a)`       | Reverses the digits of number                        |
| `Ebob(a, b)`              | Calculates GCD (Greatest Common Divisor)             |
| `ekok_bulma(a, b)`        | Calculates LCM (Least Common Multiple)               |
| `Cascad(a)`               | Provides information including digit count (example) |
